# ast3414
Python notebooks for AST3414 Cosmology, Spring 2026

# Directions

## Initial setup

### Running on Colaboratory

- Sign into your Google account
- Navigate to https://colab.research.google.com/
- Open notebook -> GitHub -> Include private repos
- Authorize connection between Colab and GitHub.
- Afterwards, assuming that your computer is able to remember you signing in, it would be possible to just click on the "Open in Colab" on the top of the notebook, or open GitHub file from Colab direcly.
- Open the notebook
- Save a copy in Drive (to ensure no changes would be lost)

### Running locally
- Download and install Anaconda https://www.anaconda.com/download
- Download a notebook from Github repository
- Open Anaconda Navigator
- Launch Jupyter Notebook or Jupyter Lab
- Open the saved notebook

## Submitting

### Colaboratory
- Save a copy in GitHub
- Select the right repository, and commit
- Submit the link to the notebook on GitHub on Canvas
  
### Anaconda
- Save the notebook
- Upload it to Github
- Submit the link to the notebook on GitHub on Canvas

## New Lab
- Check new pull requests at the beginning of the lab.
- Navigate to the pull request and merge.
- Confirm merge
- You should be able to see a new lab for the week in your repository.
